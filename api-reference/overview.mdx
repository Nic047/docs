## Imports

Mac SDK exports the following from `@mac-sdk/automation`:

```typescript
import {
  Mac,            // Main entry point
  Automation,     // Desktop mouse/keyboard control
  Key,            // Key constants (from @nut-tree-fork/nut-js)
  Notes,          // NotesApp class (alias)
  VSCodeApp,      // VS Code app class
  SystemControls, // System controls and file operations
  AppManager,     // Application manager
} from '@mac-sdk/automation';
```

## Mac class

The main entry point for all automation tasks.

```typescript
import { Mac } from '@mac-sdk/automation';
const mac = new Mac();
```

### Properties

| Property | Type | Description |
| --- | --- | --- |
| `system` | `SystemControls` | System-level controls and file operations |
| `apps` | `AppManager` | Application instance manager |
| `notes` | `NotesApp` | Built-in Apple Notes integration |
| `vscode` | `VSCodeApp` | Built-in VS Code integration |

### Methods

#### `app(name: string): BaseApp`

Get an application instance by name. Returns a cached `GenericApp` instance that inherits all `BaseApp` methods. If no app is registered under that name, a new `GenericApp` is created automatically.

```javascript
const safari = mac.app('Safari');
await safari.activate();
console.log('Is running:', safari.isRunning());
```

#### `copyToClipboard(text: string): Promise<string>`

Copy text to the system clipboard. Shortcut for `mac.system.copyToClipboard()`.

```javascript
await mac.copyToClipboard('Hello, world!');
```

#### `getClipboard(): Promise<string>`

Get the current content of the system clipboard. Shortcut for `mac.system.getClipboard()`.

```javascript
const content = await mac.getClipboard();
console.log('Clipboard:', content);
```

#### `screenshot(path?: string): Promise<string>`

Take an interactive screenshot. Shortcut for `mac.system.screenshot()`.

```javascript
const screenshotPath = await mac.screenshot('~/Desktop/screenshot.png');
```

#### `notification(title: string, message: string, sound?: string): Promise<string>`

Show a system notification. Shortcut for `mac.system.notification()`.

```javascript
await mac.notification('Task Complete', 'Automation finished!', 'Ping');
```

#### `speak(text: string, voice?: string): Promise<string>`

Convert text to speech. Shortcut for `mac.system.speak()`.

```javascript
await mac.speak('Hello!', 'Samantha');
```

#### `openApp(appName: string): Promise<string>`

Open an application by name. Shortcut for `mac.system.openApp()`.

```javascript
await mac.openApp('Safari');
```

<Info>
  For the full Mac class reference including error handling, see the [Mac class](/api-reference/mac-class) page.
</Info>

## Explorer

Access file system operations and system-level functionality through `mac.system`. The Explorer combines comprehensive file management with system controls.

### System controls

#### `screenshot(path?: string): Promise<string>`

Take an interactive screenshot using `screencapture -i`. You can capture the full screen, a window, or a selected area.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `path` | `string` | No | Path to save the screenshot (default: timestamped file in current directory) |

**Returns:** Path to the saved screenshot.

```javascript
const screenshotPath = await mac.system.screenshot();
// or specify a custom path
const customPath = await mac.system.screenshot('/Users/me/Desktop/screenshot.png');
```

#### `notification(title: string, message: string, sound?: string): Promise<string>`

Show a system notification using AppleScript.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `title` | `string` | Yes | Notification title |
| `message` | `string` | Yes | Notification message |
| `sound` | `string` | No | Sound to play (default: `'default'`) |

```javascript
await mac.system.notification('Task Complete', 'Your automation finished successfully!');
```

#### `speak(text: string, voice?: string): Promise<string>`

Convert text to speech using the macOS `say` command.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `text` | `string` | Yes | Text to speak |
| `voice` | `string` | No | Voice to use (default: `'Alex'`) |

```javascript
await mac.system.speak('Hello, this is Mac SDK speaking!');
await mac.system.speak('Welcome', 'Samantha');
```

#### `openApp(appName: string): Promise<string>`

Open an application by name using `open -a`.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `appName` | `string` | Yes | Name of the application to open |

```javascript
await mac.system.openApp('Calculator');
```

#### `setVolume(level: number): Promise<string>`

Set the system volume.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `level` | `number` | Yes | Volume level (0-100) |

```javascript
await mac.system.setVolume(50);
```

<Note>
  Throws an error if the level is outside the 0-100 range.
</Note>

#### `setBrightness(level: number): Promise<string>`

Set the screen brightness. Requires the [`brightness`](https://github.com/nriley/brightness) CLI tool to be installed.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `level` | `number` | Yes | Brightness level (0-1) |

```javascript
await mac.system.setBrightness(0.7);
```

<Note>
  Throws an error if the level is outside the 0-1 range.
</Note>

#### `sleep(): Promise<string>`

Put the Mac to sleep using `pmset sleepnow`.

```javascript
await mac.system.sleep();
```

#### `lock(): Promise<string>`

Lock the screen via CGSession.

```javascript
await mac.system.lock();
```

#### `toggleWiFi(): Promise<string>`

Toggle Wi-Fi off and back on using `networksetup`.

```javascript
await mac.system.toggleWiFi();
```

#### `getWiFiNetworks(): Promise<string[]>`

List preferred wireless networks.

```javascript
const networks = await mac.system.getWiFiNetworks();
console.log('Preferred networks:', networks);
```

#### `connectWiFi(network: string, password: string): Promise<string>`

Connect to a Wi-Fi network.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `network` | `string` | Yes | Network name (SSID) |
| `password` | `string` | Yes | Network password |

```javascript
await mac.system.connectWiFi('MyNetwork', 'my-password');
```

#### `toggleBluetooth(): Promise<string>`

Toggle Bluetooth on or off. Requires [`blueutil`](https://github.com/toy/blueutil) to be installed.

```javascript
await mac.system.toggleBluetooth();
```

#### `getDiskSpace(): Promise<object>`

Get disk usage information by parsing `df -h /`.

**Returns:** Object with `total`, `used`, `available`, and `percentage` properties (all strings).

```javascript
const disk = await mac.system.getDiskSpace();
console.log(`Disk: ${disk.used} used of ${disk.total} (${disk.percentage})`);
```

#### `getMemoryUsage(): Promise<string>`

Get memory statistics via `vm_stat`. Returns the raw output.

```javascript
const memStats = await mac.system.getMemoryUsage();
console.log(memStats);
```

#### `getCPUUsage(): Promise<string>`

Get current CPU usage from `top`. Returns the raw output.

```javascript
const cpuUsage = await mac.system.getCPUUsage();
console.log(cpuUsage);
```

#### `setDesktopWallpaper(imagePath: string): Promise<string>`

Set the desktop wallpaper using AppleScript.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `imagePath` | `string` | Yes | Absolute path to the image file |

```javascript
await mac.system.setDesktopWallpaper('/Users/me/Pictures/wallpaper.jpg');
```

### File system operations

All file system methods use Node.js `fs.promises` internally. Relative paths are resolved against `process.cwd()`.

#### `createFile(filePath: string, content?: string): Promise<string>`

Create a new file with optional content.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `filePath` | `string` | Yes | Path where the file should be created |
| `content` | `string` | No | Initial content (default: empty string) |

**Returns:** Absolute path to the created file.

```javascript
// Create an empty file
const filePath = await mac.system.createFile('new-file.txt');

// Create a file with content
const contentPath = await mac.system.createFile('readme.md', '# My Project\nThis is a sample file.');
```

#### `createDirectory(dirPath: string): Promise<string>`

Create a new directory. Creates parent directories recursively if they don't exist.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `dirPath` | `string` | Yes | Path where the directory should be created |

**Returns:** Absolute path to the created directory.

```javascript
// Create a single directory
const dirPath = await mac.system.createDirectory('my-folder');

// Create nested directories
const nestedPath = await mac.system.createDirectory('projects/my-app/src');
```

#### `readFile(filePath: string, encoding?: string): Promise<string>`

Read the content of a file.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `filePath` | `string` | Yes | Path to the file to read |
| `encoding` | `string` | No | File encoding (default: `'utf8'`) |

**Returns:** File content as string.

```javascript
const content = await mac.system.readFile('config.json');
const binaryContent = await mac.system.readFile('image.png', 'binary');
```

#### `writeFile(filePath: string, content: string, encoding?: string): Promise<string>`

Write content to a file. Overwrites existing content.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `filePath` | `string` | Yes | Path where the file should be written |
| `content` | `string` | Yes | Content to write to the file |
| `encoding` | `string` | No | File encoding (default: `'utf8'`) |

**Returns:** Absolute path to the written file.

```javascript
await mac.system.writeFile('data.json', JSON.stringify({ name: 'John', age: 30 }));
```

#### `appendToFile(filePath: string, content: string): Promise<string>`

Append content to an existing file.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `filePath` | `string` | Yes | Path to the file to append to |
| `content` | `string` | Yes | Content to append |

**Returns:** Absolute path to the file.

```javascript
await mac.system.appendToFile('log.txt', '\nNew log entry at ' + new Date());
```

#### `copy(source: string, destination: string): Promise<string>`

Copy a file or directory to a new location. Directories are copied recursively.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `source` | `string` | Yes | Path to the source file or directory |
| `destination` | `string` | Yes | Path where the copy should be created |

**Returns:** Absolute path to the copied item.

```javascript
// Copy a file
await mac.system.copy('original.txt', 'backup.txt');

// Copy a directory
await mac.system.copy('my-project', 'my-project-backup');
```

#### `move(source: string, destination: string): Promise<string>`

Move or rename a file or directory using `fs.promises.rename`.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `source` | `string` | Yes | Path to the source file or directory |
| `destination` | `string` | Yes | New path for the item |

**Returns:** Absolute path to the moved item.

```javascript
// Rename a file
await mac.system.move('old-name.txt', 'new-name.txt');

// Move to a different directory
await mac.system.move('file.txt', 'archive/file.txt');
```

#### `remove(path: string, options?: object): Promise<boolean>`

Delete a file or directory.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `path` | `string` | Yes | Path to the file or directory to delete |
| `options.recursive` | `boolean` | No | Delete directories recursively (default: `true`) |
| `options.force` | `boolean` | No | Force deletion (default: `false`) |

**Returns:** `true` if deletion was successful.

```javascript
// Delete a file
await mac.system.remove('temp-file.txt');

// Delete a directory and all contents
await mac.system.remove('old-project', { recursive: true });

// Force delete
await mac.system.remove('stubborn-file.txt', { force: true });
```

#### `listDirectory(dirPath?: string): Promise<Array>`

List the contents of a directory.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `dirPath` | `string` | No | Path to the directory to list (default: current directory) |

**Returns:** Array of `Dirent` entries with `name` property and type-checking methods like `isDirectory()` and `isFile()`.

```javascript
const contents = await mac.system.listDirectory('/Users/me/Documents');
contents.forEach(entry => {
  console.log(`${entry.name} (${entry.isDirectory() ? 'directory' : 'file'})`);
});
```

#### `exists(path: string): Promise<boolean>`

Check if a file or directory exists.

```javascript
if (await mac.system.exists('important-file.txt')) {
  console.log('File exists!');
}
```

#### `getInfo(path: string): Promise<fs.Stats>`

Get detailed information about a file or directory. Returns a Node.js `fs.Stats` object.

```javascript
const stats = await mac.system.getInfo('my-file.txt');
console.log(`Size: ${stats.size} bytes`);
console.log(`Created: ${stats.birthtime}`);
console.log(`Modified: ${stats.mtime}`);
console.log(`Is file: ${stats.isFile()}`);
console.log(`Is directory: ${stats.isDirectory()}`);
```

#### `findFiles(pattern: string, directory?: string): Promise<string[]>`

Find files matching a glob pattern.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `pattern` | `string` | Yes | Glob pattern to match (e.g., `'*.js'`, `'**/*.md'`) |
| `directory` | `string` | No | Directory to search in (default: current directory) |

**Returns:** Array of matching file paths.

```javascript
// Find all JavaScript files
const jsFiles = await mac.system.findFiles('**/*.js');

// Find all markdown files in docs directory
const mdFiles = await mac.system.findFiles('*.md', 'docs');

// Find all files with 'test' in the name
const testFiles = await mac.system.findFiles('**/*test*');
```

## Built-in apps

### Notes app

Access through `mac.notes`. The Notes app uses a folder-based API with iCloud as the default account. Note bodies are written in Markdown and automatically converted to HTML before being sent to Apple Notes.

#### `NotesApp` methods

| Method | Description |
| --- | --- |
| `iCloud` | Default `NotesFolder` instance for the iCloud account |
| `folder(name)` | Get a `NotesFolder` for a specific account name |
| `find(title)` | Search all iCloud notes for a note by exact title. Returns `{title, content}` or `null` |
| `searchNotes(query)` | Search note titles and bodies. Returns array of `{title, preview}` |

#### `NotesFolder` methods

| Method | Description |
| --- | --- |
| `open()` | Activate the Notes app. Returns `this` for chaining |
| `createNote({title, body})` | Create a new note. Body is Markdown, automatically converted to HTML |
| `createRichNote({title, body})` | Create a note with rich formatting. Body is Markdown, automatically converted to HTML |
| `appendToNote(title, content)` | Append Markdown content to an existing note by title. Returns `true` if found |
| `deleteNote(title)` | Delete a note by title. Returns `true` if found and deleted |
| `listNotes()` | List all notes. Returns array of `{title, modified, preview}` |
| `find(title)` | Search for a note by title (delegates to `NotesApp.find()`) |
| `listenForChange(title, callback, debounceTime?, pollInterval?)` | Poll a note for changes with debouncing. Returns a cleanup function |

```javascript
// Create a note with Markdown content
await mac.notes.iCloud.createNote({
  title: 'Meeting Notes',
  body: '# Agenda\n- Item 1\n- [ ] Task to do'
});

// Create a rich note (body is processed through Markdown conversion)
await mac.notes.iCloud.createRichNote({
  title: 'Custom Rich Note',
  body: '# Custom content\n- [ ] Task 1\n- [x] Task 2'
});

// List all notes
const notes = await mac.notes.iCloud.listNotes();
console.log(notes);

// Search for notes
const results = await mac.notes.searchNotes('meeting');

// Find a specific note by exact title
const note = await mac.notes.find('Meeting Notes');
if (note) {
  console.log(note.content);
}

// Append to a note
await mac.notes.iCloud.appendToNote('Meeting Notes', '\n## Follow-up\n- Review action items');

// Delete a note
await mac.notes.iCloud.deleteNote('Old Note');

// Watch for changes
const cleanup = await mac.notes.iCloud.listenForChange('My Note', (event) => {
  console.log('Changed:', event.changes.added);
});
// Later: cleanup() to stop watching
```

#### Change event object

When using `listenForChange`, the callback receives an event object with the following shape:

| Property | Type | Description |
| --- | --- | --- |
| `newContent` | `string` | The current content of the note |
| `previousContent` | `string` | The content before the change |
| `noteId` | `string` | The Apple Notes ID of the note |
| `title` | `string` | The title of the note |
| `changes.added` | `string \| null` | Text that was added |
| `changes.removed` | `string \| null` | Text that was removed |
| `changes.hasChanges` | `boolean` | Whether there are actual changes |
| `changes.position` | `number` | Character position where the change occurred |
| `changes.contextBefore` | `string \| null` | Up to 20 characters before the change |
| `changes.contextAfter` | `string \| null` | Up to 20 characters after the change |

#### Markdown conversion

Note bodies are passed through a built-in `MarkdownProcessor` that converts Markdown to HTML before sending to Apple Notes. Supported syntax:

- Headings (`# h1` through `###### h6`)
- Bold (`**text**`) and italic (`*text*`)
- Unordered lists (`- item`)
- Ordered lists (`1. item`)
- Todo checkboxes (`- [ ] task` and `- [x] done`)
- Blockquotes (`> quote`)
- Fenced code blocks (`` ```code``` ``)
- Inline code (`` `code` ``)
- Links (`[text](url)`)
- Pipe-delimited tables

### VS Code app

Access through `mac.vscode`. The VS Code app uses `open -a "Visual Studio Code"` to launch and interact with VS Code.

| Method | Description |
| --- | --- |
| `open(content?, filename?)` | Open VS Code. If content is provided, writes to a temp file and opens it. Returns the temp file path or `null` |
| `openFolder(folderPath)` | Open a folder in VS Code. Returns the folder path |
| `newFile(content, filename?)` | Create and open a new file (alias for `open()`) |
| `openWorkspace(workspacePath)` | Open a `.code-workspace` file in VS Code. Returns the workspace path |
| `createProject(name, template?)` | Create a project from a template in a temp directory and open it in VS Code. Returns the project path |

Available project templates for `createProject`:

| Template | Files created |
| --- | --- |
| `'basic'` (default) | `README.md`, `index.js` |
| `'node'` | `package.json`, `index.js`, `README.md` |
| `'web'` | `index.html`, `style.css`, `script.js` |

```javascript
// Open VS Code
await mac.vscode.open();

// Open a file with content
const tmpPath = await mac.vscode.open('console.log("hello")', 'script.js');

// Open a folder
await mac.vscode.openFolder('~/projects/my-app');

// Open a workspace file
await mac.vscode.openWorkspace('~/projects/my-app.code-workspace');

// Create a new project from template
const projectPath = await mac.vscode.createProject('my-app', 'node');
```

<Note>
  Projects created with `createProject` are placed in a temp directory (`os.tmpdir()`). Move them to a permanent location if needed.
</Note>

## BaseApp

All app instances (including those from `mac.app()`) inherit from `BaseApp`, which provides common application lifecycle methods. All methods that interact with apps use AppleScript via `osascript`. The `BaseApp` constructor takes a single `name` parameter — the exact application name as it appears in macOS.

| Method | Returns | Description |
| --- | --- | --- |
| `activate()` | `string` | Bring the app to the foreground |
| `quit()` | `string` | Quit the application |
| `isRunning()` | `boolean` | Check if the app process is running |
| `hide()` | `string` | Hide the app window (requires Accessibility permissions) |
| `show()` | `string` | Show/unhide the app (requires Accessibility permissions) |
| `minimize()` | `Promise<void>` | Minimize the front window. Falls back to Cmd+M if accessibility method fails |
| `wait(ms?)` | `Promise<void>` | Pause for `ms` milliseconds (default: 500) |
| `getWindows()` | `string[]` | Get titles of all open windows (requires Accessibility permissions) |
| `isFrontmost()` | `boolean` | Check if the app is the frontmost process |

```javascript
const safari = mac.app('Safari');
await safari.activate();

if (safari.isRunning()) {
  const windows = safari.getWindows();
  console.log('Open windows:', windows);
}

safari.quit();
```

### Internal helpers

When extending `BaseApp`, these methods are available for building custom app integrations:

| Method | Description |
| --- | --- |
| `_executeAppleScript(script)` | Execute an AppleScript string (15s timeout). Returns the result as a string |
| `_shellCommand(cmd)` | Execute a shell command (10s timeout). Returns stdout as a string |
| `_escapeAppleScript(script)` | Escape single quotes for embedding in AppleScript |
| `_escapeString(str)` | Escape quotes and newlines for embedding in scripts |

## AppManager

Access through `mac.apps`. Manages application instances and provides system-level app control.

| Method | Description |
| --- | --- |
| `get(name)` | Get or create an app instance by name. Creates a `GenericApp` if not registered |
| `register(name, appInstance)` | Register a custom app instance |
| `getRunningApps()` | List currently running `.app` processes. Returns array of `{pid, name, path}` |
| `killApp(appName)` | Kill an app by name using `pkill` |

```javascript
// List running apps
const running = await mac.apps.getRunningApps();
console.log(running);

// Kill an app
await mac.apps.killApp('Safari');

// Register a custom app
mac.apps.register('myapp', new MyApp());
```

## Automation

The `Automation` class provides low-level mouse and keyboard control via [`@nut-tree-fork/nut-js`](https://www.npmjs.com/package/@nut-tree-fork/nut-js). This is a separate class from `Mac` — use it for pixel-based desktop UI automation.

```javascript
import { Automation, Key } from '@mac-sdk/automation';

const automation = new Automation();

// Mouse and keyboard control
await automation.click(100, 200);
await automation.type('Hello, World!');
await automation.pressKey(Key.Enter);
await automation.pressKeyCombo(Key.LeftSuper, Key.Space);
```

### Constructor

```javascript
const automation = new Automation(mouseSpeed?, keyboardDelay?);
```

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `mouseSpeed` | `number` | `1000` | Mouse movement speed in pixels per second |
| `keyboardDelay` | `number` | `10` | Delay between keystrokes in milliseconds |

### Automation methods

| Method | Description |
| --- | --- |
| `click(x, y)` | Move the mouse to a pixel coordinate and left-click |
| `scrollUp(pixels)` | Scroll the mouse wheel up |
| `scrollDown(pixels)` | Scroll the mouse wheel down |
| `scrollLeft(pixels)` | Scroll the mouse wheel left |
| `scrollRight(pixels)` | Scroll the mouse wheel right |
| `type(text)` | Type a string |
| `pressKey(...keys)` | Press and release key(s). If multiple keys are passed, treats them as a combo |
| `pressKeyCombo(...keys)` | Press a key combination (e.g., Cmd+Space) |
| `releaseKey(...keys)` | Release held key(s) |
| `pause(ms?)` | Pause execution (default: 500ms) |

### Key constants

The `Key` enum is re-exported from nut.js. Common keys include:

| Key | Description |
| --- | --- |
| `Key.Enter` | Enter/Return |
| `Key.Space` | Space bar |
| `Key.Tab` | Tab |
| `Key.Escape` | Escape |
| `Key.LeftSuper` | Left Command (⌘) |
| `Key.LeftAlt` | Left Option (⌥) |
| `Key.LeftControl` | Left Control (⌃) |
| `Key.LeftShift` | Left Shift (⇧) |
| `Key.Grave` | Backtick/grave accent |
| `Key.L` | Letter L |
| `Key.N` | Letter N |

See the [nut.js Key documentation](https://nut-tree.github.io/apidoc/enums/key_enum.Key.html) for the full list.

## Extending the SDK

To add support for a new application, extend the `BaseApp` class and register it with the `AppManager`:

<Note>
  `BaseApp` is not currently exported from the main `@mac-sdk/automation` entry point. Import it directly from the subpath.
</Note>

```javascript
import { Mac } from '@mac-sdk/automation';
import { BaseApp } from '@mac-sdk/automation/core/base-app.js';

class SpotifyApp extends BaseApp {
  constructor() {
    super('Spotify');
  }

  async getCurrentTrack() {
    const script = `
      tell application "Spotify"
        return name of current track & " - " & artist of current track
      end tell
    `;
    return this._executeAppleScript(script);
  }
}

const mac = new Mac();
mac.apps.register('spotify', new SpotifyApp());

const spotify = mac.app('spotify');
const track = await spotify.getCurrentTrack();
```

See the [Integrations overview](/integrations/overview) for more details on creating custom app integrations.
