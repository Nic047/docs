---
title: Mac class
description: The main entry point for Mac SDK
---

The `Mac` class is the primary interface for interacting with Mac SDK. It provides access to system controls, built-in apps, and convenience shortcuts for common operations.

## Import

```typescript
import { Mac } from '@mac-sdk/automation';
```

## Constructor

```typescript
const mac = new Mac();
```

The `Mac` constructor takes no arguments. It initializes the system controls, app manager, and built-in app integrations (Notes and VS Code) automatically.

## Properties

| Property | Type | Description |
| --- | --- | --- |
| `system` | `SystemControls` | Access system-level controls like clipboard, notifications, and file operations |
| `apps` | `AppManager` | Manage and access application instances |
| `notes` | `NotesApp` | Built-in Apple Notes integration |
| `vscode` | `VSCodeApp` | Built-in VS Code integration |

## Methods

All methods below are convenience shortcuts that delegate to `mac.system`.

### `app(name: string): BaseApp`

Get an application instance by name. Returns a `GenericApp` (extends `BaseApp`) for any application. Instances are lazily created and cached by the `AppManager`.

```typescript
const safari = mac.app('Safari');
await safari.activate();
```

### `copyToClipboard(text: string): Promise<string>`

Copy text to the system clipboard.

```typescript
await mac.copyToClipboard('Hello, world!');
```

### `getClipboard(): Promise<string>`

Get the current content of the system clipboard.

```typescript
const content = await mac.getClipboard();
console.log('Clipboard:', content);
```

### `screenshot(path?: string): Promise<string>`

Take an interactive screenshot. If no path is provided, the screenshot is saved to the current working directory with a timestamped filename.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `path` | `string` | No | Save path for the screenshot |

**Returns:** Path to the saved screenshot file.

```typescript
// Interactive screenshot with default path
const screenshotPath = await mac.screenshot();

// Custom path
const customPath = await mac.screenshot('~/Desktop/screenshot.png');
```

### `notification(title: string, message: string, sound?: string): Promise<string>`

Display a macOS system notification.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `title` | `string` | Yes | Notification title |
| `message` | `string` | Yes | Notification message |
| `sound` | `string` | No | Sound to play (default: `'default'`) |

```typescript
await mac.notification(
  'Task Complete',
  'The operation finished successfully',
  'Ping'
);
```

### `speak(text: string, voice?: string): Promise<string>`

Convert text to speech using the macOS `say` command.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `text` | `string` | Yes | Text to speak |
| `voice` | `string` | No | Voice to use (default: `'Alex'`) |

```typescript
await mac.speak('Hello, I am your Mac assistant', 'Samantha');
```

### `openApp(appName: string): Promise<string>`

Open an application by name using the macOS `open -a` command.

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `appName` | `string` | Yes | Application name |

```typescript
await mac.openApp('Safari');
```

## Error handling

Methods that execute AppleScript or shell commands throw standard `Error` objects when they fail. Wrap calls in try/catch for robust error handling.

```typescript
try {
  await mac.openApp('NonExistentApp');
} catch (error) {
  console.error('Failed:', error.message);
}
```

## Example

```typescript
import { Mac } from '@mac-sdk/automation';

async function main() {
  const mac = new Mac();

  // Take a screenshot
  const screenshotPath = await mac.screenshot('~/Desktop/desktop.png');
  console.log(`Screenshot saved to: ${screenshotPath}`);

  // Create a note
  await mac.notes.iCloud.createNote({
    title: 'Automation Log',
    body: 'Script executed successfully.'
  });

  // Show notification
  await mac.notification('Done', 'Automation complete!');
}

main().catch(console.error);
```
