This guide walks you through creating a simple automation script with Mac SDK.

## Prerequisites

- Node.js 14.0.0 or later
- macOS 10.15 (Catalina) or later
- Xcode Command Line Tools (install with `xcode-select --install`)

<Note>
  Mac SDK is an ES module package. Your project must use `"type": "module"` in `package.json` or use the `.mjs` file extension.
</Note>

## Installation

Install the Mac SDK package using npm:

<Tabs>
  <Tab title="npm">

    ```bash
    npm install @mac-sdk/automation
    ```
  </Tab>

## Your first script

Create a new file called `automate.js`:

```javascript automate.js
import { Mac } from '@mac-sdk/automation';

async function main() {
  // Initialize the Mac SDK
  const mac = new Mac();

  try {
    // Open Safari
    console.log('Opening Safari...');
    await mac.openApp('Safari');

    // Create a new note in the iCloud Notes account
    console.log('Creating a note...');
    await mac.notes.iCloud.createNote({
      title: 'My First Note',
      body: '# Hello from Mac SDK!\n\nThis note was created automatically.'
    });

    // Take a screenshot
    console.log('Taking a screenshot...');
    const screenshotPath = await mac.screenshot('~/Desktop/mac-sdk-demo.png');

    // Show a notification
    await mac.notification(
      'Automation Complete',
      'Your Mac SDK script ran successfully!',
      'Ping'
    );

    console.log(`Screenshot saved to: ${screenshotPath}`);
  } catch (error) {
    console.error('An error occurred:', error);
  }
}

// Run the script
main();
```

## Running the script

```bash
node automate.js
```

## Troubleshooting

### Missing permissions

If you encounter permission errors, grant the necessary permissions in:

<Steps>
  <Step title="Open system settings">
    Navigate to **System Settings** > **Privacy & Security** (macOS 13+) or **System Preferences** > **Security & Privacy** > **Privacy** (macOS 12 and earlier).
  </Step>
  <Step title="Add your terminal app">
    Add your terminal app (e.g., Terminal or iTerm) to:
    - **Accessibility**
    - **Screen Recording**
    - **Automation**
  </Step>
</Steps>

### Module not found

If you see `Cannot find module '@mac-sdk/automation'`:

1. Run `npm install` in your project directory.
2. Verify the package name in your `package.json`.
3. Check your Node.js version with `node -v` (should be 14.0.0 or higher).

### ERR_REQUIRE_ESM

If you see `ERR_REQUIRE_ESM`, Mac SDK is an ES module and cannot be loaded with `require()`. Either:

1. Add `"type": "module"` to your `package.json`.
2. Rename your script file to use the `.mjs` extension.
