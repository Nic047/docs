## What you can automate

Mac SDK gives you programmatic control over macOS through a clean JavaScript API. Here are the main categories of automation you can build.

### System control

- Adjust volume, brightness, and wallpaper
- Take screenshots and trigger notifications
- Monitor CPU, memory, and disk usage
- Manage Wi-Fi and Bluetooth connections
- Lock, sleep, or wake your Mac

### File management

- Create, read, write, copy, and move files
- Search files with glob patterns
- Automate backup routines
- Organize downloads by file type

### Application automation

- Launch, quit, hide, and show any macOS app
- Check if apps are running or frontmost
- Get window titles and manage app lifecycle
- Build custom integrations by extending `BaseApp`

### Desktop automation

- Click, scroll, and type at pixel-level precision
- Press key combinations and shortcuts
- Automate multi-step UI workflows
- Combine with app control for end-to-end automation

### Apple Notes

- Create notes with Markdown content (auto-converted to HTML)
- Search, list, and delete notes
- Append content to existing notes
- Watch notes for real-time changes with diff detection

### VS Code

- Open files, folders, and workspaces
- Scaffold new projects from built-in templates
- Combine with `Automation` class for full IDE automation

## Real-world use cases

### Automated development environment

- Scaffold project directories with `mac.system` and open in VS Code
- Run tests and capture screenshots of results
- Notify when builds complete or fail

### Personal knowledge base

- Save code snippets to Apple Notes with language and description
- Search and organize notes programmatically
- Watch shared notes for changes in real-time

### System monitoring

- Track CPU, memory, and disk usage over time
- Generate system reports and save to Notes
- Get notifications when resources are running low

### Workspace management

- Switch between "work" and "relax" app configurations
- Adjust volume and open the right set of apps
- Close unnecessary apps automatically
